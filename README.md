# A uniform driver for CAN device 

### Device supported

  - NI
    - USB-8473

  - ZLG(周立功) 
    - USBCAN-I/II
    - USBCANFD-200U
    - USNCANFD-400U(only channel 1 and channel 2 can be used)
    - USBCANFD-800U

### Welcome to contribute for adding more device supported.
